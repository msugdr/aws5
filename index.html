折り返し制御

word-break:break-all

問答無用で折り返します。
（例）
QWERTYUIOPASDFGHJKLZXCVBNMqwertyuiopasdfghjklzxcvbnmQWERTYUIOPASDFGHJKLZXCVBNMqwertyuiopasdfghjklzxcvbnm

word-break:keep-all

区切り以外では折り返さないスタイルです。禁則処理を守ります。
（例）
日本語の場合も読点や句点および空白あるいは括弧などがなければ「折り返す」ことなく表示するスタイルです。句点があったの折り返されるはずです。

overflow-wrap:break-word

禁則処理をできるだけ守るが、非常に長いものは折り返す。

空白と折り返し制御

通常は半角の空白は１つにまとめられます。これを変更することおよび改行制御も同時に するのが white-space です。

white-space:pre

空白・改行をそのまま維持する。折り返しはしない。<pre>タグと同じ
（例）
function myFirstFunction (boid) { for (int i; i<10; i++) { if (x = classX.getValue() ) { classX.setValue("LOVE")} else { classX.setValue("HITE")}; } }

white-space:pre-wrap

空白・改行をそのまま維持して折り返す。<pre>タグにスタイルの上書きができる
（例）
function myFirstFunction (boid) {
    for (int i; i<10; i++) {
        if (x = classX.getValue() ) { classX.setValue("LOVE")} else { classX.setValue("HITE")};
    }
}

文字の配置

文字の配置は text-align で左寄せ（標準）中央 (center)、右寄せ (right) があるが、均等などもある。

text-align:justify

インラインコンテンツは両端揃えされます。テキストは最終行を除いて、その左右の端が行ボックスの左右の端に揃うように間隔が空けられます。最終行を除いてというのがポイントです。最終行は改行タグなどで判断しているようです。
